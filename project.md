# 封装 API

    为什么获取个人信息里面不需要传个人 id？

        因为登录的时候，获取到的token其实已经包含了登录的用户的信息了，所以调用接口的时候，带上token的就已经相当于把登录用户的id带上了，服务会根据带上的token去获取登录的用户的信息，而获取token本身也是一个获取用户信息的接口
